# Архів хендофів

Одноразові записки минулих сесій («що зробила конкретна сесія»). Зберігаються довідково —
**можуть бути застарілі**: актуальні правила та карта системи живуть у `CLAUDE.md`,
`PROJECT_MAP.md`, README пакетів і `docs/infrastructure/`.

Правило: новий `HANDOFF-<тема>.md` — тимчасовий; довговічні факти одразу переносяться в
канонічні доки, а сам файл після відпрацювання переїжджає сюди (див. `HANDOFF.md` у корені).

> `HANDOFF-vps-deploy.md` містить секрети первинного розгортання — він у `.gitignore`
> і існує лише локально.
