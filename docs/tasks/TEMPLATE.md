## Реалізація

Що фактично зроблено + ключові рішення/трейдофи. Дотримано правил із `CLAUDE.md`:

- [ ] i18n: нові тексти додано і в укр, і в EN-словник (`web/src/lib/i18n.tsx` / `bot/i18n.ts`); перевірено дублі (`uniq -d`, інакше TS1117)
- [ ] Tailwind v4: нові класи виписані буквально (`web/src/lib/colors.ts`)
- [ ] Ролі/доступ: мапа узгоджена `api-server/src/lib/roles.ts` ↔ `web/src/lib/roles.ts`; фінанси — owner-only
- [ ] Схема БД (якщо змінювалась): SQL накатано через `psql`; `pnpm run typecheck:libs`
